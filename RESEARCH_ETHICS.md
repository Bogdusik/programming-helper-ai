# Research Ethics Compliance Guide

## Overview
This application is designed for academic research on AI-assisted programming learning. All data collection must comply with ethical research standards and GDPR requirements.

## Data Collection Policy

### ✅ What IS collected (anonymized):
- Anonymous user interactions with the AI
- Programming questions and AI responses
- Usage statistics (anonymized)
- Learning progress metrics

### ❌ What is NOT collected:
- Personal email addresses
- Identifiable user information
- Real names or personal details
- Any data that could identify participants

## Implementation Details

### 1. Anonymous User IDs
- Users are identified only by anonymous Clerk-generated IDs
- No email addresses are stored in the database
- All data is linked to anonymous identifiers only

### 2. Research Consent
- All participants must provide explicit consent
- Consent is obtained before any data collection
- Participants can withdraw at any time
- Clear information about data usage is provided

### 3. Data Retention
- Data is retained only for research purposes
- Participants can request data deletion
- No personal information is ever collected

## For Researchers

### Data Analysis
When analyzing collected data:
1. Use only anonymous participant IDs
2. Aggregate data to prevent individual identification
3. Report only statistical patterns, not individual responses
4. Ensure all findings are anonymized

### Participant Rights
- Right to withdraw consent
- Right to data deletion
- Right to information about data usage
- Right to access their anonymized data

## Legal Compliance

This implementation ensures compliance with:
- GDPR (General Data Protection Regulation)
- University ethical guidelines
- Academic research standards
- Data protection best practices

## Contact

For questions about data handling or participant rights, contact the research team.
